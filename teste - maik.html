Conta palavras
Desafio N:1

Crie uma função que receba uma string e retorne um Hash com a quantidade de ocorrências de cada palavra.
ATENÇÃO: Não usar a função split, ou outra que automatize o processo de quebra da string em palavras. Exemplo: contaOcorrencias('asa bola casa asa bola asa') teria como saída um hash com a seguinte estrutura:
{
         'asa': 3,
         'bola': 2,
         'casa': 1
}
